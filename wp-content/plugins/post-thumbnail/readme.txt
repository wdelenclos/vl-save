The plugin displays posts' thumbnail in dashboard "posts list" table.

== Description ==

The plugin adds posts' thumbnail in dashboard "posts list" table and allows to check which post has a thumbnail or not.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-thumbnail` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Post Thumbnail screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin add thumbnails on custom post types? =

Yes! The plugin supports custom post types. You must go to plugin settings page and just check the post type you want to add the thumbnails.

= Post type not appears in settings page =

This will happen only if the post type does not support thumbnail feature

Post type supports: <https://codex.wordpress.org/Function_Reference/post_type_supports>
